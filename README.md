# 📊 Java Basic Array Programs

This repository contains basic Java programs focused on **array concepts**.  
It helps in understanding how arrays work and how to perform operations like searching, storing, and processing data using arrays in Java.

---

## 📌 Programs Included

The repository includes the following array-based programs:

- 🔍 Linear Search (Q_Search_Num.java)
- 📊 Marks Storage using Array
- 📈 Marks Processing (marks_2.java)
- 📏 Array Size Handling (size.java)

---

## 🛠 Technologies Used

- **Java**
- **Arrays**
- **Loops**
- **Conditional Statements**
